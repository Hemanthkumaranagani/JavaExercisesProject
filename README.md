# Java Exercises

## Programs

### PrimeNumberChecker
Checks whether a number entered by the user is prime or not.

### FibonacciGenerator
Generates the Fibonacci sequence for the number of terms entered by the user.

## How to Run

1. Open the project in IntelliJ IDEA.
2. Run PrimeNumberChecker.java.
3. Enter a number when prompted.
4. Run FibonacciGenerator.java.
5. Enter the number of terms when prompted.
